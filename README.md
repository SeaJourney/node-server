# Node.js Server
This is a Node.js server which runs on the localhost and lets you run a website on your local computer.

# Installation

Install the [latest release](https://github.com/SeaJourney/node-server/releases/tag/v1.0.1) and extract the files. For this you also need [Node.js](https://nodejs.org/en/download) to execute this server!
#### Dependencies:
```node
npm i chalk@4.1.2
```
```node
npm i clear@0.1.0
```
```node
npm i express@4.18.2
```
```node
npm i fs@0.0.1
```
